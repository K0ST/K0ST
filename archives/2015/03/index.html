#

弹窗信息

是一款让你在Android设备上拥有ios通知功能的软件，除此之外还有很多个性化设置，在功能和体验上更加完善。在各大市场均可下载。

下载量：500万次（免费付费各大市场总和）

累计用户：70万人（按独立设备统计）

日弹窗量：200万次

技术创新：

  1. 仿照系统源码重新开发设置界面(K-PreferencScreen)，使其更加灵活多变；
  2. 主题切换功能（包括皮肤、自定义颜色等等）；
  3. 可视化投票模块（使用RecyclerView等），让用户可以发布自己的投票表达意见。

魅手机隐私锁

是一款独立设计与开发的手机锁定软件，可以锁定App与文件等。七大锁定模式：九宫格锁定模式、手势锁定模式、密码锁定模式、程序假死锁定模式、提示锁定模式、不执行模式、程序跳转模式。每个模式的作用具体是什么还需要您自己来体会。文件隐藏：不仅仅可以隐藏android系统下面的文件，而且当手机作为u盘连接到windows电脑上面时，文件也会被隐藏的天衣无缝。

下载量：12万次（魅族市场收费版）

技术难点：

  1. 防止用户破解，保证锁定的坚固有效；
  2. 密码等隐私信息的存储；

计划派

如果你正被每天琐碎的种种急得焦头烂额，如果你曾因为忘了准备礼物而让TA生气，如果你渴望把一切打理的井井有条，那么“计划派”正是你的不二选择！她将为你提供：最简单易用的计划管理列表；最轻松自如的交互操作；最简洁大方的用户界面。她让繁杂的事物变得亲切有序，让远大的目标变得真实可企及。

下载量：5万次

技术难点：

  1. 列表的拖动与排序功能；
  2. 自定义时间Picker的效率优化与时间展示问题。

夜间模式

你是否有晚上玩手机的习惯？是否觉得最小亮度还是刺眼？《夜间模式》可帮您将亮度降到更低！保护您的双眼！并具备手电筒功能哦~

下载量：16万次（魅族市场收费版）

技术实现：本软件实现较为简单，使用WindowManager创建覆盖整个屏幕的悬浮窗即可。仅使用了两天开发的时间，但是还取得了比较好的销量，着实比较意外。

大象框

大象框是赵云科技的一款新产品，简单来说是一个云电子相框，可以通过手机向异地的长辈传送照片，免去让他们操作复杂手机的烦恼。我在其中独自负责相框端软件的开发。

官方网站：大象框

技术难点：

  1. 低端平板上大量高清图片的展示与存储（需要良好的内存管理避免OOM）；
  2. 低端平板上交互的重构与流畅度优化。
